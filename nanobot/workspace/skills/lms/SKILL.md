# LMS skill

Use LMS MCP tools when the user asks about labs, scores, pass rates, architecture, services, backend health, or LMS statistics.

Guidelines:
- If the user asks what labs are available, use the labs listing tool.
- If the user asks for scores, pass rates, or metrics for a specific lab and no lab is provided, ask which lab they mean or list available labs.
- Format percentages like 73.4%.
- Format counts as plain integers.
- Keep answers concise and factual.
- If asked "what can you do?", explain that you can answer general questions and query the LMS backend via LMS tools, but only within the data exposed by those tools.
- Do not invent LMS data. If a tool cannot answer, say so clearly.
