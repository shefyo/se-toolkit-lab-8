## What is a terminal

A terminal (or terminal emulator) is a program that provides a text-based interface for interacting with the [operating system](./operating-system.md#what-is-an-operating-system).

It runs a [shell](./shell.md#what-is-a-shell), which accepts commands and displays their output. When you type a command and press the `Enter` keyboard button, the shell interprets the command and the terminal displays the result.

An example of a terminal is the [`VS Code Terminal`](./vs-code.md#vs-code-terminal).
