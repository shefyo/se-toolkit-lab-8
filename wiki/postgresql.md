## What is `PostgreSQL`

`PostgreSQL` (also known as `Postgres`) is a popular open-source relational [database server](./database.md#database-server).

Docs:

- [Official `PostgreSQL` docs](https://www.postgresql.org/docs/)

## `PostgreSQL` server

The `PostgreSQL` server is the background [service](./operating-system.md#service) process that manages [databases](./database.md#database) and handles queries from [clients](./http.md#web-client).

## `PostgreSQL` server in this project

In this project, `PostgreSQL` runs as a [`Docker` container](./docker.md#container).

See [`PostgreSQL` with `Docker`](./docker-postgres.md#what-is-postgresql-with-docker) for how to manage it in this project.
